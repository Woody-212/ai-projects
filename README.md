## 📊 Stock Price Analyser

Pulls real-time stock market data and visualises price trends using moving averages.

**What it does:**
- Fetches 6 months of historical stock data via Yahoo Finance
- Calculates 20-day and 50-day moving averages
- Plots and saves a chart for visual trend analysis

**Tech used:** Python, pandas, matplotlib, yfinance

**How to run:**
```bash
pip install yfinance matplotlib pandas
python stock_analyser.py
```

**Why it matters:** Moving averages are one of the most widely used indicators 
in quantitative finance — the same logic underpins algorithmic trading systems 
used by fund administrators and hedge funds globally.
